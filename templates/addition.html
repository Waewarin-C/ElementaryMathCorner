Addition Page

What is Addition?

Addition is when we take two or more numbers and joining it together to make a bigger number.

When we read or say the addition problem we say, "A number plus another number equals a bigger number".

Different Parts of an Addition Problem

The numbers being added together are called addends.

The answer to an addition problem is called a sum or total.

Note: The order of the numbers don't matter when adding. For example, 7 + 4 is the same as 4 + 7

Adding Using Columns

Another way to write an addition problem is to write the numbers from top to bottom instead of from left to right. When we do that, the numbers will make a column. This can be used to add as many numbers as we want.

To add numbers using columns, we do

  1. Write the numbers from top to bottom
  2. Write the plus sign on the left side of the numbers
  3. Draw a horizontal line under the last number
  4. Then we just add the numbers and write the sum below the line.

This is also a common way to add numbers with two or more digits.

Adding Numbers with Two or More Digits

To add numbers with two or more digits, we will add using columns. To do this, write one number below the other, while making sure the place values line up.

Then add the numbers one column at a time.

What Happens When there are Numbers that have More Digits Than Others?

When that happens, don't panic, we just write the numbers the same way as when both numbers have the same number of digits.

Then add the numbers one column at a time

For the columns that don't have any numbers to add, we simply bring it down. We can also think of it as the number plus an invisible 0.

Regrouping

Sometimes when we add numbers in a column, the sum is greater than 10. When that happens we do what is called regrouping.

This can happen on any column or on more than one column in an addition problem.

How to Regroup

  1. Take the sum and regroup into the tens and ones places
  2. Write down the ones digit
  3. Write the tens digit on top of the next column. This is called carrying over.
  4. Move on to adding the next column. Don't forget to add the number that was carried over.
  5. If the sum for that column is greater than 10, repeat the steps.
