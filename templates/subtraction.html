Subtraction Page

What is Subtraction?

Addition is when we take two or more numbers and joining it together to make a bigger number.

When we read or say a subtraction problem we say, "A number minus another number equals a smaller number".

Different Parts of a Subtraction Problem

The number that is being subtracted from is called the minuend.

The number that is doing the subtracting is called the subtrahend.

The answer to a subtraction problem is called a difference.

Note: The order of the numbers matter when subtracting. For Example, 9 - 3 is NOT the same as 3 - 9

Subtracting Using Columns

Like addition, we can write a subtraction problem from top to bottom instead of from left to right. When we do that, the numbers will make a column. This can be used to subtract as many numbers as we want.

To subtract numbers using columns, we do the same thing as addition except we write a minus sign and the answer is the difference.

This is also a common way to subtract numbers with two or more digits.

Subtracting with Two or More Digits

To subtract numbers with two or more digits, we will use columns. To do this, write one number below the other, while making sure the place values line up.

Then subtract the numbers one column at a time.

What Happens When there are Numbers that have More Digits Than Others?

Don't worry, just like with addition we write the numbers the same way as when both numbers have the same number of digits.

Then subtract the numbers one column at a time

For the columns that don't have any numbers to subtract, we simply bring it down. We can also think of it as the number minus an invisible 0.

Borrowing

Sometimes when we subtract numbers in a column, the top number is smaller. When that happens we do what is called borrowing.

This can happen on any column or on more than one column in a subtraction problem.

How to Borrow

  1. Borrow from the next column. For example, if we're on the ones column, we borrow from the tens columns, and if we're on the tens column, we borrow from the hundreds column.
  2. When we borrow, we are always borrowing 10, so we add 10 to the number that is borrowing.
  3. Cross out original number and write the new number on top. This is now the number we are going to be subtracting from.
  4. Do the subtraction for the column.
  5. Move on to subtracting the next column.
  6. If the top number for that column is smaller, repeat the steps.
